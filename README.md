## Standardization Format

This is a temporary idea for standard input for variance estimation using `sys` given a set of arbitrary (plot-level) attributes.

```
x    - x Coordinate
y    - y Coordinate
z_1  - First attribute
...
z_R  - Rth attribute
pi_i - The first order inclusion probability
```
